# todo-app-with-reactnative

## アプリの概要

React Nativeで作ったtodoアプリです。
Youtubeの動画　(https://youtu.be/VhiH1votwm8?feature=shared)　を参考にして開発しました。
todoの追加と編集、削除が可能です。
